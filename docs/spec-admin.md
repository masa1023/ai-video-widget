packages/adminはウィジェットで再生する動画等を管理するためのアプリケーションです。
主な機能は以下の通りです。

- 新規登録、ログイン、パスワードリセット等のユーザー認証機能 (auth)
- プロジェクト/サイトの管理 (projects)
  - プロジェクト/サイトの登録・編集・削除
  - scriptタグの発行
- 動画のアップロード (videos)
- 動画の付随情報の管理 (slots)
- スロット間の遷移定義 (slot_transitions)
- コンバージョンルールの定義 (conversion_rules)
- ダッシュボード表示 (dashboard)
  - 動画の再生状況
  - クリック状況
  - コンバージョン状況
